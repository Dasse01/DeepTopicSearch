📚 심화탐구주제검색기

맥에듀테크 · 김휘창입시컨설팅 | 합격생의 심화탐구보고서 주제 검색기
로딩중...
1
🔍 심화탐구 주제 검색
상단 검색창에 키워드를 입력하면 제목·대학·학과·교과목 전체에서 즉시 필터링됩니다. 예) "면역", "미적분", "기후변화"
각 칼럼 헤더 아래 입력창으로 대학·학과·주제·교과목·활동유형·탐구방법을 따로 세부 필터할 수 있습니다. 여러 칼럼 동시 필터링도 가능합니다.
2
⭐ 즐겨찾기 추가
마음에 드는 주제 행 맨 왼쪽의 ☆ 별 버튼을 클릭하면 즐겨찾기에 저장됩니다. 다시 클릭하면 해제됩니다.
상단 ⭐ 즐겨찾기 탭에서 저장된 목록을 카드 형식으로 확인하고, AI 보고서 생성도 이 탭에서 시작합니다. 여러 주제를 미리 저장해 두세요.
3
🔑 Claude API 키 설정
즐겨찾기 탭 상단의 🔑 API 키 설정 버튼을 클릭합니다. console.anthropic.com에서 회원가입 후 API 키(sk-ant-...)를 발급받아 입력하세요.
키는 이 세션에서만 사용되며 외부에 저장되지 않습니다. 키 입력 후 🔍 확인 버튼으로 정상 여부를 테스트할 수 있습니다.
4
🤖 AI 탐구보고서 자동 생성
즐겨찾기 카드의 🤖 AI 보고서 생성 버튼을 클릭하면 4단계(서론→본론→결론→키워드·참고문헌)로 자동 작성됩니다. 약 30~60초 소요됩니다.
생성된 보고서는 초안으로만 활용하고, 자신만의 탐구 과정·데이터·해석을 추가해 완성도를 높이세요.
5
📄 PDF / 워드 저장
AI 보고서 화면 상단 툴바의 📄 PDF 저장을 클릭하면 A4 형식으로 인쇄·저장됩니다. 표지·서론·본론·결론·참고문헌이 페이지별로 구분됩니다.
📋 워드/한글 복사는 서식 포함 클립보드 복사로, Word·HWP에 붙여넣기하면 서식이 그대로 유지됩니다.
6
🔥 인기 소재 & 교과목별 탐구
상단 🔥 인기 소재 TOP 20 탭에서 전체·인문·자연 계열별로 가장 많이 등장한 탐구 주제 20선을 확인할 수 있습니다.
📋 교과목별 탐구 탭에서는 수강 중인 교과목을 클릭하면 해당 교과목과 연계된 모든 탐구 주제가 자동 검색됩니다. 주제 발굴의 출발점으로 적극 활용하세요.
7
📊 주제 빈도 분석 (워드클라우드)
인문/자연계열 버튼 선택 후 학과 드롭다운에서 원하는 학과를 고르면, 그 학과 합격생들이 자주 사용한 심화탐구 키워드가 워드클라우드로 시각화됩니다.
글자가 클수록 빈도 높은 핵심 키워드입니다. 단어 클릭 시 전체 검색 탭으로 자동 이동합니다. 하단 빈도 순위 표에서 상위 30개 키워드와 비율을 순서대로 확인하세요.
💡
필터 초기화: 검색 패널 상단 🔄 초기화 버튼으로 모든 필터를 한 번에 해제할 수 있습니다.  |  특수계열 필터: 의예과·치의예과·약학 등은 상단 주황색 버튼으로 빠르게 필터링하세요.  |  인문/자연 분류: 전체·인문계열·자연계열 탭으로 계열별 주제만 볼 수 있습니다.
